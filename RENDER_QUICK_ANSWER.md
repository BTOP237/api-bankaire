# ⚡ RENDER - Configuration Rapide (Copier-Coller)

## 🎯 Voici EXACTEMENT ce qu'il faut mettre

### 1️⃣ Répertoire racine
```
.
```

### 2️⃣ Chemin du Dockerfile
```
./Dockerfile
```

### 3️⃣ Variables environnementales

Ajouter ces deux variables:

```
APP_ENV = production
APP_DEBUG = false
```

---

## ✅ C'EST TOUT!

Le Dockerfile s'occupe du reste automatiquement. ✨

---

## 📝 Notes

- ✅ Le fichier `Dockerfile` est dans votre repository
- ✅ Le port `8080` est configuré dans le Dockerfile
- ✅ Composer installer les dépendances automatiquement
- ✅ La base de données SQLite est créée automatiquement

---

## 🚀 Après avoir cliqué "Create"

1. Attendre 2-3 minutes
2. Chercher le message **"Live"** en vert
3. Copier l'URL: `https://banking-api-xxxxx.onrender.com`
4. Tester: `curl https://banking-api-xxxxx.onrender.com/api/users`

**C'est en ligne!** 🎉
